什么是MybatisPlus（MP）

MyBatis-Plus（简称 MP）是一个 MyBatis 的增强工具，在 MyBatis 的基础上只做增强不做改变，为简化开发、提高效率而生。

MP的特性

这里只学习在SpringBoot下的MP配置和使用方法

  1. 在创建SpringBoot项目时，勾选上mybatisplus(在阿里提供的spring网址中才有)

别忘了去配置application.properties!!!

  1. 创建dao层接口，并添加Mapper注解

也可以不在这里添加@Mapper注解，而是在SpringBoot的运行类类名上添加:@MapperScan()注解，自动进行mapper包扫描

  1. BookDao继承的BaseMapper接口中，已经提前写好了很多sql语句，可以直接进行使用

wrapper中的基本比较操作

  • eq：等于
  • ne：不等于
  • gt：大于
  • ge：大于等于
  • lt：小于
  • le：小于等于
  • between：在值1到值2之间
  • notBetween：不在值1到值2之间
  • in：字段in(“字段名”, value0, value1…)
  • not in：字段not in(“字段名”, value0, value1…)

模糊查询

  • like：like ‘%值%’
    • eg: like(“name”, “王”) => name like ‘%王%’
  • notLike：和上面相反
  • likeLeft：like ‘%值’
    • eg: likeLeft(“name”, “王”) => name like ‘%王’
  • likeRight：like ‘值%’
    • eg: likeRight(“name”, “王”) => name like ‘王%’

排序

  • orderBy：orderBy(true, true, “id”, “name”) —> order by id ASC,name ASC
    • true代表ASC【升序】，false代表DESC【倒序】
  • orderByAsc：orderByAsc(“id”, “name”) —> order by id ASC,name ASC
  • orderByDesc：orderByDesc(“id”, “name”) —> order by id DESC,name DESC

逻辑查询

  • or：主动调用 or 表示紧接着下一个方法用 or 连接(不调用 or 则默认为使用 and 连接)
  • and：默认
